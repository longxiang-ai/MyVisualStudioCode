公式计算ver1.0说明：
1.字符串第一个字符为= + -时认定为表达式模式
2.已经可以支持形如 B1, B1:H1 格式数据的读取和公式计算，可以实现如=sum(A1,B1:C4,3)的数据计算
3.暂时将公式返回值统一为double类型，左下角状态栏支持展示当前选中的单元格（如A1  A1:B2），显示当前单元格内的公式
4.支持运算符： + - * /  >= <= > < ^  %(单目运算符)
5.支持符号：,(函数中参数分隔符)  :（函数中引用单元格表示区域）
6.支持数值类型：整数、浮点数、科学计数法、百分数、分数
7.支持函数：SUM AVERAGE COUNT MAX MIN FACT ABS LOG LN SIN COS TAN ASIN ACOS ATAN
8.尚未实现的功能：gui界面的函数计算（如excel一样设置按钮），当函数公式间存在嵌套循环调用的情况判断 ......
9.尚未兼容合并单元格后的情况
